# SISANTRI

SISANTRI adalah aplikasi pengelolaan santri berbasis Laravel untuk presensi sholat, presensi diniyah, rekap kehadiran, dan pengaturan pengguna.

## Fitur Utama

- Presensi sholat dengan QR dan input manual
- Presensi diniyah dan tahfidz
- Rekap bulanan diniyah dan sholat
- Pengelolaan data santri
- Pengelolaan pengguna dan hak akses

## Menjalankan Proyek

1. Install dependency PHP dan JavaScript.
2. Salin `.env.example` menjadi `.env` jika belum ada.
3. Jalankan migrasi database.
4. Jalankan server Laravel dan Vite.

## Catatan

- Akun pengguna dibuat oleh admin.
- Tampilan sistem sudah dioptimalkan untuk desktop dan mobile.
